## 说明

这是一个react的ssr项目，基于以下 ：

- react16.8+ (support hooks)
- koa2
- webpack4.x
- react-redux7.2+ (support hooks)

特性:

- 开发模式支持热更新
- 基于路由的代码分割
- 自定义Head以支持SEO
- 数据预加载（js/css）

## 使用方法

### `npm install`

安装依赖

### `npm run dev`

开发模式运行<br>
地址： [http://localhost:3000](http://localhost:3000)

### `npm run build`

生产模式打包

### `npm run start`

运行打包后资源
